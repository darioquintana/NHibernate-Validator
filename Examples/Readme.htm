NHibernate Validator Quickstart

-

Welcome to the NHibernate Validator Quickstart !

The NHibernate Validator Team had prepare examples to you be able to use this validation framework in a easy way. To learn more about NHibernate Validator visit NHibernate Forge.

Our wiki and blog

Examples:

1) Winforms

In this example you will how you can easily integrate NHibernate Validator and some helper clases with System.Windows.Forms.

2) Integration

Trying to integrate NHibernate Validator with NHibernate? Not a problem. In this example you will how really get these 2 great frameworks working together. NHibernate Validator let you intercept Saves and Updates from NHibernate validating your entities before these changes take place.

3) Asp.Net MVC

Asp.Net MVC has a cool way to add validation errors from model and display them all into the View. This example cover the easy integration between these frameworks. Note, you need Asp.Net MVC installed into your machine. Download Asp.net MVC
